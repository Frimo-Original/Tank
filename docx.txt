drawelements.h - вспомогательная библиотека для удобства отрисовки.

Функции:

setColor(int r, int g, int b) - метод для установки цвета заливки и об-
водки фигур.

drawButton(int x, int y, int x2, int y2, char nameButton[]) - метод для
рисования кнопок. Сама кнопка серого цвета, а надпись в ней красного.
Первые два аргумента функции - координата верхнего левого углакнопки,
а вторые два - координата нижнего правого угла кнопки. Последний аргумент
надпись на кнопке.

drawPanel(int width, int height) - рисует своеобразную панель белого
цвета от верхнего левого угла окна, до точки переданной в параметрах
функции.

drawImage(int x, int y, int width, int height, HDC image) - функция, ри-
сующая картинку переданную в параметрах функции. Координаты рисования
картинки, также передаются в параметрах.

cleanField(int x, int y, int x2, int y2) - "отчищает" заданную область
фона в параметрах функции, если фон чёрного цвета.

drawLabel(int x, int y, int x2, int y2, char text[]) - рисует надпись,
по координатам переданных в параметрах функции. Фон надписи белый, а сама
надпись чёрная. Последний параметр текст надписи.
